# 09 — Dynamic Programming

DP takes the most time to get comfortable with. Follow the order below.

## Resources

- Standard DP: GFG articles + **Aditya Verma** (YouTube) — primary resource
- AtCoder Educational DP Contest — 26 problems, highly structured
- SPOJ — clean DP problems
- Digit DP: Codeforces blog
- Bitmask DP: HackerEarth blog
- DP on Trees: GFG articles + Rachit Jain (YouTube)
- SOS DP: cp-algorithms blog

## Checklist

- [ ] All standard DP problems from GFG (with Aditya Verma)
- [ ] AtCoder Educational DP Contest — all 26 problems
- [ ] SPOJ DP problems
- [ ] Codeforces DP problem list (search "dynamic programming codeforces blog")
- [ ] Digit DP — understand recurrence, solve problems
- [ ] DP with Bitmasks
- [ ] DP on Trees
- [ ] SOS DP (Sum over Subsets)
