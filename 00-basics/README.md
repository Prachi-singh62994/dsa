# 00 — Basics

Before anything else. These build your intuition for loops, arrays, and thinking algorithmically.

## Checklist

- [ ] Pattern printing problems (Sourabh Shukla — YouTube)
- [ ] Time complexity analysis — Big O, don't overthink it, learn by doing contests
- [ ] Linear search & circular array representation (any article, no practice needed)
- [ ] Palindrome, Perfect numbers, Armstrong numbers
- [ ] Hashing / frequency counting
- [ ] Prefix Sum — 1D and 2D
- [ ] Sliding Window technique

## Resources

- Pattern problems: Sourabh Shukla on YouTube
- Prefix Sum & Sliding Window: CP Sheet — https://bit.ly/tuf_CPList
