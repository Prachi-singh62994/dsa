# Dynamic Programming

## Key Concepts
- Overlapping subproblems + optimal substructure
- Top-down: memoization (recursion + cache)
- Bottom-up: tabulation (iterative)
- Common patterns: 0/1 knapsack, longest subsequence, grid DP
