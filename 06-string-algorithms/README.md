## Reference

| Algorithm | Use Case | Time |
|-----------|----------|------|
| Rabin-Karp | Pattern matching using hashing | O(n+m) avg |
| KMP | Pattern matching without backtracking | O(n+m) |
| Z-function | Pattern matching, string periodicity | O(n+m) |
| Manacher | Longest palindromic substring | O(n) |
| Rolling Hash | Substring comparison, multiple queries | O(n) |
