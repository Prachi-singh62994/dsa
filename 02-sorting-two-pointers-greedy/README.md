## Sorting Algorithms

| Algorithm | Time (avg) | Space | Stable? | Status |
|-----------|-----------|-------|---------|--------|
| Bubble Sort | O(n²) | O(1) | Yes | ⬜ |
| Selection Sort | O(n²) | O(1) | No | ⬜ |
| Insertion Sort | O(n²) | O(1) | Yes | ⬜ |
| Merge Sort | O(n log n) | O(n) | Yes | ⬜ |
| Quick Sort | O(n log n) | O(log n) | No | ⬜ |
